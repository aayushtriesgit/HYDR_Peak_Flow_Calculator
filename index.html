Method & equations (SCS / NRCS — TR‑55 Ch. 2 and LA DOTD Hydraulics Manual Ch. 1)

This calculator reproduces the computation performed by LaDOTD program HYDR1130 (hydr2009 package, rev. 1998‑07‑14). It is a small storm‑hydrograph model, not the TR‑55 Exhibit 4 “graphical peak discharge” chart, so results are the same as HYDR1130 but may differ slightly from TR‑55 chart solutions.

1. Runoff depth — SCS runoff curve‑number method (TR‑55, eqs. 2‑1 to 2‑4):

S = 1000/CN − 10    Iₐ = 0.2·S    Q = (P − 0.2S)² / (P + 0.8S)  for P > Iₐ, else Q = 0

2. Time of concentration — SCS watershed‑lag method (TR‑55 eq. 3‑4) with ℓ = hydraulic length (ft), Y = average watershed slope (%):

lag (hr) = ℓ0.8 · (S + 1)0.7 / (1900 · √Y)    T꜀ = lag / 0.6

3. Storm distribution — the 24‑hr design storm is distributed with the SCS Type II cumulative mass curve (the table embedded in HYDR1130), and incremental runoff is computed with the CN equation over unit periods of duration D = T꜀/7.5.

4. Unit hydrograph & convolution — each unit period’s incremental runoff is routed with the SCS triangular unit hydrograph (peak factor 484, time to peak Tₚ = D/2 + 0.6·T꜀, recession 1.67·Tₚ, tabulated at 13 ordinates). The composite hydrograph is the sum of all increments; the reported value is its peak multiplied by the Peak Adjustment Factor Fₚ:

qₚ = Fₚ · maxt Σ ΔQⱼ · UH(t − tⱼ)    where the UH peak = 484 · A(mi²) · 1″ / Tₚ

Input limits enforced by HYDR1130 (mirrored here): drainage area ≤ 2,000 acres (800 ha), watershed slope 0.1 % – 30 %.

TR‑55 cautions (Ch. 2): curve numbers describe average conditions for design use; the CN equation contains no time/intensity term; the Iₐ = 0.2S assumption was generalized from agricultural watersheds — use judgment in heavily urbanized cases.

Fidelity to HYDR1130

This page is an independent re‑implementation of HYDR1130’s SCS routine, reverse‑engineered from the program and verified against 82 batch runs of the original executable spanning drainage area 200–64,000 ac, length 200–70,800 ft, CN 40–95, rainfall 1–20 in, slope 0.1–30 % and Fₚ 0.5–100. All cases matched to the precision printed by the program (typically better than 0.01 %).

One quirk of the original program is reproduced intentionally: HYDR1130 searches for the hydrograph peak starting at the 12ᵗʰ ordinate of the composite hydrograph. For very long times of concentration (T꜀ ≳ 12.5 hr) the true hydrograph peak occurs earlier than that, so the program (and this page) reports a value slightly below the true computed peak. When this happens a note is shown beneath the results with the unclipped value.
